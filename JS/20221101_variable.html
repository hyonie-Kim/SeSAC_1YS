변수 - var, let, const

  • 자바스크립트는 변수를 선언할 때 사용하는 키워드로 총 3가지가 있다. - var, let, const
  • var는 ES6(ES2015)표준이 발표된 이후로 부터는 많이 사용하지 않는다.
    이유는 변수를 선언한 위치와 상관없이 전역으로 사용할 수 있기 때문에 어디에서 선언 되었는지, 호출 했는지 위치를 명확히 알수 없다.
  • 브라우저에서는 기본적으로 window 객체를 사용한다.

🌱 let은 언제 사용하고, const는 언제 사용하는가?

  • let은 어떠한 변수 값의 변경이 빈번하게 일어날 때 사용하는 것이 좋다.
  • const는 값이 변화되지 않기 때문에 예측이 가능하다.

🌱 let과 const는 용도에 명확하게 사용해야 한다.

  • let과 const는 블록 단위이기 때문에 블록 안에서 작성된 변수는 블록 밖에서 호출하면 호출을 할수가 없다.
  • 그렇지만 블록 밖에서 선언된 변수는 블록 안으로 타고 내려 갈수가 있기때문에 호출이 가능하다.
                
                    예제1]

                    {
                        let eng ="scalper!!"
                        console.log(eng)
                    }

                    예제2]
                    
                        let accessibility ="접근하다"
                    {
                        
                        console.log(accessibility)
                    }

데이터 타입

  • typeof 명령을 사용하면 데이터 타입이 어떠한 형태 인지 알수 있다.
  1. String
  2. Number
  3. Boolean
  4. null - 빈값
  5. undefined - 값이 선언되지 않은 상태
  6. symbol
            
                예제]
                const name = "hyonie";
                const age = 33;
                const isMale = false;
                const isfemale = true;
                let money = null; 
                let boyFriend = undefined;

                console.log(typeof name)

문자열 사이에 변수를 넣고 싶을 때 사용하는 방법..?

  • 문자열과 변수를 같이 사용하고 싶을 경우에는 기호 + 표시를 해줘야 한다.
  • ES6 이후로는 템플릿 문법인 백텍(`)기호를 사용해서 불필요한 더하기 기호 사용하지 않고 한번에 출력한다. 
            
                console.log("my name is" + name + "and i am " + age)

                console.log(`내이름은 ${name}이고 나이는 ${age} 입니다.`)

🌱 문자열을 자르고 싶을때 사용하는법 - substring

            
                예제1] ti 출력
                const animal = "tiger";
                console.log(animal.substring(0,2))

                예제2]  ger을 출력 - (2,5)는 2번째 인덱스 글자부터 5번째 인덱스 앞의 글자까지 출력
                console.log(animal.substring(2,5))

🌱 문자열을 쪼개는 방법 - split

            
                let fruits = "사과, 딸기, 포도, 참외";
                
                예제1] 배열로 나누기

                
                fruits = fruits.split(", ");
                console.log(fruits);

                예제2]배열중에 2번 출력
                console.log(fruits[2]) //포도

배열과 오브젝트

  • 여러가지 값들을 묶어놓은 자체를 배열이라고한다. 
            
                const animals = ["lion", "rabbit", "monkey"];

                예제1] 3번째 인덱스에 새로운 값 넣기
                animals[3] = "tiger"
                animals.push("tiger")

                예제2] 배열 맨앞에 새로운 값 넣기 - 0번째 인덱스
                animals.unshift("dog")

                예제3] 배열 앞에 있는 값을 빼낼때 - 맨뒤에 있는 값이 빠짐
                animals.pop()

                console.log(animals);

배열인지 확인하는 메서드 Array.isArray()

  • isArray 메서드를 사용하면 true 또는 false 를 반환한다.
    배열인 경우에는 true, 아닌 경우에는 false를 반환한다. 
            
                const fruits = "apple, banana, strawberry";

                console.log(Array.isArray(fruits))

특정 값의 요소에 인덱스를 알고 싶을 때, 사용하는 메서드- indexOf()

            
                console.log(animals.indexOf("monkey")); //2

A~C까지 값을 지정해서 강제로 빼는 방법 - splice, slice

  • splice는 인자를 2개 넣는다.(선택 인덱스 번호, 빼고싶은 개수)
            
                const animals = ["lion", "rabbit", "monkey", "dog", "cat"];

                예제] lion, rabbit 빼기
                animals.splice(0,2) // ["monkey", "dog", "cat"]
                animals.splice(1,2) // ["lion", "dog", "cat"]
                animals.splice(1,3) // ["lion", "cat"]

                예제2] indexOf 메서드 활용 
                animals.splice(animals.indexOf("monkey"), 3) // ["lion", "rabbit"]

                console.log(animals)
  • slice는 splice와 차이가 있다.
  • splice는 배열 자체를 바꾸고, slice는 배열을 바꾸지 않고 값만 반환한다.
    때문에 변수 선언을하고 값을 담아야한다.
  • 잘린 값만큼만 출력된다
            
               const slicedAnimals = animals.slice(0,2);
               console.log(slicedAnimals); // ["lion", "rabbit"]
