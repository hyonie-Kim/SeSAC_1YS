오브젝트

  • 오브젝트는 배열하고 비슷하면서도 약간다르다.
  • 오브젝트는 속성과 값, 즉 쌍을 이루고 있는 데이터이다.
  • 배열을 값으로 가지는 키도 선언할수 있다.
  • 오브젝트 안에 또 다른 오브젝트를 선언할 수도 있다.
        
            const monkey = {
                name : 'kiki',
                age : 5,
                weight: 40,
                gender: 'male',
                isCute: true,
                
                food: ["banana", "carrot", "potato"]
                
                home: {
                    location: "zoo",
                    size: "big",
                    price: 3000
                }
            }

            console.log(monkey);

JSON(Javascript Object Notation) - 데이터를 주고 받을때(전송)

  • 참조 사이트 구글에서 "Json Formatter"검색, Json의 형식이 올바른 형식으로 변형이 되었는지 확인해주는 사이트
  • Json의 특징은 key와 value 오브젝트 형식으로 되어 있지만, Json형식은 Key가 쌍따옴표로 감싸져 있다.

🌱 JSON.stringify(monkey) 메서드 - 오브젝트가 JSON 형태로 문자열화 된다

      
        const monkeyJSON = JSON.stringify(monkey);
        console.log(monkeyJSON);

🌱 JSON.parse(monkey) 메서드 - 오브젝트 형태로 파싱을 시켜준다

      
        const parsedJSON = JSON.parse(monkeyJSON);
        console.log(parsedJSON);
