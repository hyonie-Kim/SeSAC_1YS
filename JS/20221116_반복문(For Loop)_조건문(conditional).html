조건문

  • if문 괄호 안에는 조건이 들어간다.
  • 이퀄 표시가 2개(==)인것과 3개(===)인 것 차이점
    • == :데이터타입 비교안함
    • === :데이터타입 까지 비교
  • if와 else중간에 else if 를 넣어 또 다른 조건문을 만들수 있다.
            
            const num = 9;
            
            예제1]

            if(num < 10){
                console.log('10보다 작습니다.')
            }else {
                console.log('10보다 크거나 같습니다.')
            }

            예제2]
            if(num < 10){
                console.log('10보다 작습니다.')
            } else if (num === 10){
                console.log('10과 같습니다.')
            } else {
                console.log('10보다 큽니다..')
            }

switch문

  • switch 소괄호에는 비교할 값을 넣어준다
  • 각각 case마다 비교할 값을 넣어주면된다.
  • 명령이 끝나면 break를 작성하여 switch문이 종료되었다는 걸 표시한다.
  • default는 if else에서 else와 같은 역할을한다.
            
            const animal = 'monkey';
            // const animal = animals[2].name
            switch(animal){
                case 'lion':
                    console.log('사자입니다.');
                    break;
                case 'monkey':
                    console.log('원숭이입니다.');
                    break;
                default:
                    console.log('동물이 아닙니다.');
                    break;
            }

반복문

  • 소괄호에 반복의 조건을 넣는다.
  • i가 최초의 0으로 선언이 되었고, i가 3보다 작을 때까지 반복된다. i가 한번 반복될때마다 1씩증가
    값이 3일때, 3보다 작지않으니까 반복문이 종료된다. 
        
            for(let i=0; i < 3; i++){
                console.log(i);
            }

배열의 길이만큼 반복하기

  • 예제2번에 animal에 각각의 요소들이 하나씩 담기고 animals에 있는 길이만큼 출력한다.
  • 보통 배열을 선언할 때 s를 붙여 복수 형태로 선언을 한다. 안에 인자들은 단수 형태로 선언한다.
        
            예제1]
            for(let i=0; i < animals.length; i++){
                console.log(animals[i]);
            }

            예제2]
            for(let animal of animals) {
                console.log(animal)
            }
            예제3] animal 이름만 출력
            for(let animal of animals) {
                console.log(animal.name)
            }
